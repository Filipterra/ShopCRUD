Welcome to simple frame of a CRUD webapp for managing a shop.

Installation and running:
1. Extract this ZIP archive into your desired directory.
2. Make sure you have the latest version of Maven and Java.
3. Go to parent directory of the project (the one where this file was) and open the Terminal.
4. Run "mvn install" and "mvn package".
5. Navigate to the "target" directory.
6. Run "java -jar alpha-0.0.1-SNAPSHOT.jar".
7. Open an Internet browser and go to localhost:8080 to see the results.

Known issues:
If you encounter strange errors popping out in the top-right corner try a diffrent browser or clear your cashe.
